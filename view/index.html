Meteo API

This is the public API for the Meteostation at the Kantonsschule Glarus.
If you are looking for the current data of the Meteostation, go to meteo.kanti-glarus.ch.
If you are looking for some awesome JSON-outputs, you are right here!

API Access

Current Values as JSON

Links

  • /api/1.0/json/single/temperature
  • /api/1.0/json/single/air_humidity
  • /api/1.0/json/single/air_pressure
  • /api/1.0/json/single/rainfall
  • /api/1.0/json/single/snow
  • /api/1.0/json/single/sunlight
  • /api/1.0/json/single/sun_uv
  • /api/1.0/json/single/wind_speed
  • /api/1.0/json/single/wind_speed_max
  • /api/1.0/json/single/wind_direction
  • /api/1.0/json/single/wind_deviation

Example

    Request: /api/1.0/json/single/air_temperature

    Response: {
        "air_temperature": "11.1",
        "id": "86729",
        "date": "2020.10.07 20:10"
    }

Values over time as JSON

Links

  • /api/1.0/json/multiple?start=1601942400&stop=1602055060
  • /api/1.0/json/multiple?start=1601942400&stop=1602055060&delta=60

Example

    Request: /api/1.0/json/multiple?start=1601942400&stop=1602055060&delta=60
    start: Timestamp / start of the period
    stop: Timestamp / stop of the period
    delta: time-delta of the entries in minutes
    hint: when time-entries are missing, delta jumps over it - no correction.

    Response: {
        {
            "id": "86611",
            "date": "2020.10.06 03:40",
            "air_temperature": "9.6",
            "air_humidity": "94.7",
            "air_pressure": "1013.9",
            "sunlight": "-0.7",
            "sun_uv": "0.0",
            "wind_speed": "0.4",
            "wind_speed_max": "1.3",
            "wind_direction": "331",
            "wind_deviation": "14.3",
            "rainfall": "0.0",
            "snow": "0.0"
        }, {
            "id": "86602",
            "date": "2020.10.06 02:40",
            ...
